📌 Halo 프로젝트 상세

구현목적

채권관리 솔루션을 이용 중인 고객사에서 추심(전자소송)을 진행해야하는 경우가 있는데, 채무자의 개인정보나 전자소송에 필요한 이미지 파일등이 솔로션에 관리는 돼있지만 전자소송사이트에 입력을 해야하는 번거로움이 있음.

또한 전자소송사이트에 올라오는 송달문서파일을 솔루션에 수기로 업로드를 하고있어서 불편함 호소.

언어 / 프레임워크 / DB
  • PHP
  • Python
  • Java
  • SpringBoot
  • FastApi
  • Laravel
  • Postgresql
구현기능
  • 전자소송사이트 스크래핑을 통한 송달문서 적재 / 전자소송 진행(윈도우서버)
  • 솔루션서비스와 연동을 위한 RESTAPI구현(Fastapi)
  • 문서 OCR 처리(Naver Clova OCR) 및 데이터 적재(Laravel)
  • 채권관리 솔루션 연동 페이지 및 비즈니스 로직 구현(Laravel)
  • 요금산정을 위한 관리자페이지 사용건수 체크용 API구현(Springboot)
주요 문제 및 해결
  • 카드결제 진행시 보안프로그램 우회 및 랜덤키패드 처리
  • Naver clova OCR 동시요청 제한
